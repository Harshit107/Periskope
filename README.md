# Periskope UI

**Manage your WhatsApp groups and chats at scale.**

Periskope is a multi-number, multi-agent shared WhatsApp inbox that helps teams collaborate, integrate systems, and automate workflows effectively. This repository contains the frontend UI for the Periskope application, featuring a responsive dashboard for managing WhatsApp groups, chats, and contacts.
